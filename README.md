\# Real-Time Face Detector using OpenCV



\## About

This project captures live webcam video and detects faces in real time

using OpenCV's Haar Cascade Classifier in Python.



\## Technologies Used

\- Python 3.x

\- OpenCV (cv2)

\- Haar Cascade Classifier



\## How to Run



1\. Install dependencies:

pip install opencv-python



2\. Run the script:

python face\_detector.py



3\. Press 'q' to quit the webcam window.



\## How it Works

\- Captures each frame from the webcam

\- Converts it to grayscale

\- Runs Haar Cascade face detection on each frame

\- Draws green rectangles around detected faces
